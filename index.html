PCT-100 化工过程控制实训装置

基于 ESP32 的物联网智能控制平台

产品概述

PCT-100型化工过程控制实训装置是一套基于 ESP32 微控制器的物联网智能控制平台，集成温度传感器、光照传感器、继电器控制、WiFi通信、MQTT协议等功能模块，提供完整的工业控制实训环境。

技术参数

  • 主控: ESP32-C3 Dev Module
  • 电源: DC 5V / USB-C
  • 通信: WiFi 802.11 b/g/n
  • MQTT: 支持 MQTT 5.0 协议

系统组成

  • 温度传感器：DS18B20 (GPIO 10)
  • 光照传感器：光敏电阻 (GPIO 1)
  • 继电器输出：4路继电器控制
  • OLED显示：128×64 I2C屏幕
  • LED指示：WS2812 RGB灯带
  • 按键输入：总闸控制键

核心功能

🌡️

温度监测

DS18B20数字温度传感器，实时采集环境温度，支持阈值越限报警

💡

光照检测

光敏电阻模拟采集，教材公式换算为 lux 值，支持双阈值滞回控制

⚡

继电器控制

4路继电器输出，控制灯光（继电器3）和风扇（继电器4）

🌐

WiFi连接

自动连接WiFi，支持串口配置网络，自动重连机制

📡

MQTT通信

支持远程控制和状态上报，主题格式：chemctrl/PCT_100_xxx

🔄

模式切换

自动模式（传感器阈值控制）/手动模式（按键控制）

工作模式

🔧 自动模式

系统根据传感器采集数据自动控制执行器：

  • 温度 > 32°C 时，自动开启风扇
  • 光照 < 260 Lux 时，自动开启灯光
  • 支持双阈值滞回算法，防止频繁切换

🎮 手动模式

用户通过按键手动控制执行器状态：

  • 短按切换继电器状态组合（4种状态）
  • 按住1.5秒切换工作模式
  • 支持远程MQTT控制

MQTT通信协议

状态上报主题

chemctrl/PCT_100_{device_id}

命令下发主题

chemctrl/PCT_100_{device_id}/command

状态上报格式

{
  "temperature": 27.35,
  "light": 453,
  "mode": "auto",
  "key_lock": true,
  "relay3": false,
  "relay4": false,
  "temp_threshold": 32,
  "light_threshold": 260
}

命令下发格式

{
  "cmd": "set_relay",
  "relay": 3,
  "value": true
}
